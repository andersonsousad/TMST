```markdown
# Task Management System for Teams

## Description
A complete task management system for teams, featuring a RESTful API, back-end in Python, and front-end in React and Angular.

## Technologies Used
- **Back-end:** Python, Flask, MySQL
- **API:** .NET, REST
- **Front-end:** React, Angular

## Repository Structure
- **backend/**: Contains the code for the back-end server and business logic.
- **api/**: Contains the code for the RESTful API built with .NET.
- **frontend/**: Contains the code for the user interfaces, built with React and Angular.

## How to Run

### Backend (Python/Flask)
1. Navigate to the `backend` directory and follow the instructions in the specific README.

### API (.NET)
1. Navigate to the `api/TaskManagementAPI` directory and follow the instructions in the specific README.

### Frontend (React/Angular)
1. Navigate to the `frontend/react-app` and `frontend/angular-dashboard` directories and follow the instructions in their respective READMEs.

## Running the Complete System

1. Start the back-end as per the instructions in `backend/README.md`.
2. Start the API as per the instructions in `api/README.md`.
3. Start the front-end as per the instructions in `frontend/react-app/README.md` and `frontend/angular-dashboard/README.md`.
```
